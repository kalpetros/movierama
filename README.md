# MovieRama

## Getting started

To run MovieRama type

```
$ yarn develop
```

and visit **http://localhost:1234**

or use the generated **index.html** file located in the **/dist** folder.

To run a production build type:

```
$ yarn build
```

Run tests:

```
$ yarn test
```

To run code formatting type:

```
$ yarn format
```

For linting type:

```
$ yarn lint
```
